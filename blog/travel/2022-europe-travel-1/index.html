[2022 유럽] #1 Tomorrowland를 향해!

꿈에만 그리던 투모로우랜드 티켓팅 성공 그리고 준비.

업데이트됨: 2023년 6월 16일 오전 02:34

EDM 덕후들의 버킷리스트 Tomorrowland

저는 평소 EDM을 즐겨 듣는 편입니다. 사실 그냥 즐겨 듣는다고 하기엔 많이많이 즐겨 듣습니다..!!😁 그래서 항상 언젠간 Tomorrowland나 Ultra Miami같은 메이저 페스티벌에 가겠다는 꿈을 마음 한편에 가지고 있었습니다. 그런데 마침 코로나 기세가 잠깐 꺾여서 그런지 그간 멈춰 있던 Tomorrowland를 개최한다는 소식이 나왔고 또 이때가 아니면 언제 유럽여행을 갈지도 모르겠어서 겸사겸사 Tomorrowland 티켓팅을 도전해보게 되었습니다.

티켓팅 사전 등록 성공!

올해 드디어 3년 만의 오프라인 투모로우랜드 개최를 위한 티켓팅 사전 등록을 2월 쯤 진행한다고 해서 저도 한 번 도전해보기로 했습니다. 항상 너무 가보고 싶었던 곳이라 재미 삼아 한 번 해보는거였지만 괜히 긴장되서 웃겼던 것 같습니다.😋 그런데 생각지도 못하게 우리나라 FIRST 20 안에 들게 되었습니다!(좀 많이 진심이었구나 실제 우리나라에서는 얼마나 가는지 궁금하기는 합니다.🤔)

투모로우랜드는 국가별로 사전 등록에 성공한 선착순 20명에게 티켓팅을 조금 더 편하게 할 수 있는 기회를 제공해줍니다. 찾아보기로는 나중에 따로 티켓팅을 한다면 전세계 사람들과 동시에 티켓팅을 하게 되어서 성공하기가 꽤 힘들다고 합니다..! 결국 좋은 기회를 얻게 되어 고민한 끝에 티켓팅을 하기로 마음먹고 절반은 투모로우랜드를 위한😋 유럽여행 계획을 세우게 되었습니다.

어떤 티켓을 사야할까?

티켓팅을 앞두고서 어떤 티켓을 사야할 지 고민이 많이 되었습니다. 투모로우랜드 티켓은 여러 옵션에 따라 종류가 나뉘기 때문에 자신의 상황에 맞게 적절한 티켓을 고르면 됩니다.

  • Day Pass - 1일권

  • Full Madness Pass - 3일권(or VIP존 입장 가능한 Full Madness Comfort Pass)

  • DreamVille Package - 드림빌(투모로우랜드 캠핑 에리어) 숙박 + 3일권

  • Global Journey Travel Package - 이동편(버스, 기차, ...) + 숙박(드림빌, 호텔, ...) + 3일권

※ 세부정보는 여기서 확인해주세요!

크게 이런 식으로 티켓이 나뉘고 다시 세부적으로 티켓이 나뉩니다. 저는 3일권이 포함된 티켓을 고르려고 했는데, 몇몇 블로그들을 찾아보니 DreamVille Package를 가장 많이 추천하셔서 저 안에서 고르기로 결정을 했습니다!

드림빌에서 어디에 묵을까?

DreamVille은 투모로우랜드 부지 내에 마련된 자체 캠프 에리어입니다. 안에는 텐트나 숙소들이 종류별로 나누어져 있어서 만약 DreamVille Package를 골랐다면 어떤 텐트나 숙소에서 묵을 지 결정을 하면 됩니다.

  • MAGNIFICENT GREENS AREA - 직접 텐트를 가져와서 설치

  • CAMP2CAMP AREA - 미리 설치된 텐트 제공

  • FRIENDSHIP GARDEN AREA - 10명 단위 그룹을 위한 공간 제공

  • EASY TENT AREA - 각종 편의 기능(에어 매트리스, 침낭, 전등, 텐트 락커, ...)이 포함된 텐트 제공

  • MONTAGOE AREA(💸💸💸)

  • MOUSAI AREA(💸💸💸💸💸💸💸...)

※ 세부정보는 여기서 확인해주세요!

저는 친구랑 둘이 같이 가는데 이런저런 편의를 생각했을 때 EASY TENT 2인용이 좋겠다고 생각중이었습니다. 그런데 찾아보니 EASY TENT 중에서 Spectacular EASY TENT가 외부 바닥이 우드 데크로 되어 있어서 혹시 비(그 근방이 날씨 변덕이 심하다고 해요...)가 내리더라도 침수 위험이 적고 220V 콘센트가 텐트 안으로 들어온다고 해서 조금 더 비쌌지만 Spectacular EASY TENT로 최종 선택을 했습니다!

나중에 알게 된 것 인데 페스티벌이 모두 끝나면 텐트를 포함한 제공된 모든 것은 가지고 올 수 있어요!!(텐트는 접이식이긴 한데 비행기 수하물로 부치기가 애매할 것 같아서 저희는 텐트만 빼고 다 가져왔습니다.😁)

드디어 실전 티켓팅!

드디어 실전 티켓팅날이 되었고 마음속에 티켓까지 결정해두어 무척 설렜습니다! 일단 20명끼리 경쟁하는거라곤 하지만 얼마나 치열할지 감이 안잡혀 조금 긴장 했었습니다▪▪▪ 지금와서 기억에 남는 것들을 조금 적어보면 다음 같은 것들이 있었습니다.

  • 티켓팅 사이트 접속 후 대기 중 절대 새로고침 하지 않기!

    • 외부 티켓팅(결제??) 모듈을 연결해서 쓰는 것 같은데 새로고치면 큐에서 맨 뒤로 밀린다고 하더라구요..!😥

  • 카드 해외 이용 차단 여부 확인하고 미리 풀어두기!

이 정도만 기억하셔도 큰 무리 없이 원하시는 티켓을 구매하실 수 있으실꺼에요! 물론 마음에 드는 티켓을 미리 정해두고 실전에서 빠르게 잘 클릭하는게 중요하겠죠!⏳

티켓팅을 성공하면 기다리는 일들

마침내 투모로우랜드 티켓을 손에 거머쥐었습니다!😭 항상 마음 속으로만 생각하던 곳을 드디어 가게 되어서 무척 두근두근거렸습니다.😲 티켓팅에 성공한다면 다음과 같은 일들이 기다리고 있답니다.

  1. 티켓의 구매가 확정됨(personalized) 👉 특별한 상황이 아니면 이제 환불은 불가능!!

  2. Treasure case(입장을 위한 ID팔찌 등이 든 패키지) 배송을 위한 주소 입력 이메일이 옴

  3. 투모로우랜드 계정에서 E-ticket을 다운로드 받을 수 있는 링크가 활성화 됨

  4. 6월 쯤 Add-on voucher를 구매할 수 있음(개인 락커, 주차, 식사 예약, ...)

⚠️ 투모로우랜드 입장을 위해서는 저 Treasure case 안에 든 ID팔찌가 무조건 필요합니다!! 출국 전에 잘 받을 수 있도록 출국일을 잘 정하시고 배송지 주소도 꼭 더블 체크하세요!

저도 걱정이 되어 투모로우랜드 측에 문의를 해보았는데 혹시나 팔찌를 못 받고 가더라도 행사장에서 해결은 해주는 듯 하기는 하니 너무 걱정할 필요는 없는 것 같습니다!

이렇게 우선 복잡하고도 스릴넘쳤던(?) 티켓팅이 일단락 되었습니다. 기다려라 투모로우랜드 내가 간다~~~~
